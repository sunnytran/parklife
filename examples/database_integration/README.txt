How to Install


1. Make sure you have thparkdb loaded in your postgres program, and have node.js installed
2. In the terminal, go to the directory where you saved the server.js and json files
3. Enter 'npm init', make sure to set the entry point to server.js
4. Now do 'npm install' to get the libraries needed
5. In the server file, change the database password to the password you have
(Change any other settings that are different too!)
6. In the terminal, enter 'node server.js' to run the program.


An example query should show in the terminal if done correctly
